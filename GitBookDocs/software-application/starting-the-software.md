# Starting the Software

Starting the ONTOS _CLEAN_ software is a straightforward process. Follow these steps to initiate the software:

{% hint style="success" %}
1. Navigate to the Windows Desktop.
2. Locate the ONTOS _CLEAN_ software icon.
3. Double-click on the icon to launch the software.
{% endhint %}

Upon completing these steps, the ONTOS _CLEAN_ software interface will be launched.
